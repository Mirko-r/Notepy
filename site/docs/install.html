How to install Notepy

Linux

  • Install Python
  • $ git clone https://github.com/Mirko-r/Notepy.git && cd Notepy
  • $ pip install -r requirements .txt
  • $ python main.py

Windows

    Build from source

  • Install Python
  • $ git clone https://github.com/Mirko-r/Notepy.git && cd Notepy
  • $ pip install -r requirements .txt
  • $ python main.py

    Download exe

  • Download from Github release page
