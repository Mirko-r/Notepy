# INSTALLATION

<h3>LINUX</h3>

```
  $ git clone https://github.com/Mirko-r/Notepy.git && cd Notepy
  $ pip install -r requirements .txt
  $ python main.py
```
<h3>WINDOWS</h3>

Download exe from <a href="https://github.com/Mirko-r/Notepy/releases/">release page</a>
