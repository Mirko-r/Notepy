Notepy is a fully-featured Notepad Python app

<h2 align="center">INSTALLATION</h2>
<h3>LINUX</h3>

```
  $ git clone https://github.com/Mirko-r/Notepy.git && cd Notepy
  $ pip install -r requirements .txt
  $ python notepy.py
```
