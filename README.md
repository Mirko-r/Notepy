## Setup

```shell
$> git clone https://github.com/Mirko-r/Notepy && cd Notepy
$> pip install -r requirements.txt
$> python3 notepy.py
```

**You can also download Notepy from the** [Release page](https://github.com/Mirko-r/Notepy/releases)
